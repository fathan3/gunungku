Gunungku PHP Native (Fixed Final for XAMPP)

Cara pakai:
1. Extract folder ini ke C:\xampp\htdocs\gunungku
2. Import file gunungku_xampp_database.sql ke phpMyAdmin
3. Jalankan Apache dan MySQL
4. Buka http://localhost/gunungku/

Akun demo:
- radit@gunungku.id / 123456
- admin@gunungku.id / 123456

Catatan:
- Seed password di SQL memakai placeholder, jadi login demo menerima password 123456.
- User baru dari register akan memakai password hash normal.
